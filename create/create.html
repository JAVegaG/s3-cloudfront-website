Since DynamoDB is a NoSQL database, one is not bound by a particular structure when creating a new record. However, DynamoDB is a key-value database and since it requires a partition key, any item must have it. In this particular case, the partition key is an id. To add a field, click the plus sign and to remove a field, click the minus sign.
